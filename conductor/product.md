# Product Definition: lifx-async

## Project Name

lifx-async

## Description

A modern, type-safe, async Python library for controlling LIFX lights.

## Problem Statement

There's no modern, type-safe Python library for local LIFX device control with zero runtime dependencies.

## Target Users

- Python developers building home automation and smart lighting applications
- Developers integrating LIFX lights into async Python projects (e.g., Home Assistant)

## Key Goals

- Zero runtime dependencies
- Full async/await support via Python's built-in asyncio
- Strict type safety with Pyright validation
- Auto-generated protocol support from official LIFX specifications
